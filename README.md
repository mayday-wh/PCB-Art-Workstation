# PCB Art Assistant | PCB 艺术加工助手

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Python](https://img.shields.io/badge/python-3.8+-green.svg)

在翻看一些喜欢的漫画和浮世绘时，经常会冒出一个念头：如果这些线条不只是存在于屏幕或纸张上，而是通过沉金、喷锡、绿油和丝印，永远‘刻’在电路板（PCB）上，会是什么样子？

想法付诸实践，有了这个下面的项目，一个将普通位图转化为 PCB 物理层的 PC 端小工具。它可以帮助你将普通的彩色图片转化为符合 PCB 生产工艺（如丝印、阻焊、铜箔层）的黑白分色图。

从 [Releases](../../releases) 页面下载最新版本的 `.exe` 文件，双击即可直接运行。使用方法见下文的功能模块介绍。

## 🌟 功能模块介绍

1.  **色卡录入**

要把一张颜色复杂的图片变成 PCB 上的图层，首先需要知道PCB能实现哪些颜色，所以第一步就是录入每种阻焊和展示模式的色卡。

首次运行软件，会在根目录生成colors.json，用于存储色卡信息，色卡绑定阻焊颜色和展示模式。我提供的软件压缩包，包含了我录入了部分色卡信息的colors.json文件。

色卡可以自行修改，颜色可以从颜色盘选取，也可以从导入图片选取，只要对应的层设置正确即可。

如下图所示，我从EDA软件的截图中选色，并定义了颜色对应的层，然后录入该色卡。

2.  **色彩聚集**

在新的色彩聚类模块中，整合了之前的降噪合并以及添加标定的功能。

选择阻焊颜色，有/无背光，即可导入模块1中录入的相应色卡，然后只需在导入图片中选取你分别想要映射的颜色。

如果原图片比较单调，色卡中用不到的颜色无需映射，空着即可。误选的话，可以点击提取色卡重新映射，或者再次点击映射重选颜色。

预览效果基本与成品差不多，点击导出图纸会自动导出物理层的图纸。未涉及的层不会导出，图纸带有编号，对应关系如下：TS-顶层丝印, TM-顶层阻焊, TL-顶层, BL-底层, BM-底层阻焊, BS-底层丝印。

折腾过 PCB 艺术画的朋友都知道，多图层导入 EDA 时，对齐位移简直是噩梦。我添加了一个‘<b><font color="#673AB7">原点标定</font></b>’的小功能。

在色彩聚类模块左侧栏位，可以看到原点标记设置，可选择在图片的角落（可多选，通常左上就可以）添加了一个默认长度是图片 1/100 宽度的三角形锚点，它刚好藏在 PCB 板框那 2mm 的圆角里。在 EDA 里导入时，锁住原点（0，0）。成品出来后，这些标记点又会被倒角完美切除，不留痕迹。

如果你也正好喜欢绘画，又恰巧是个电子爱好者，希望这个工具能帮你省去一些繁琐的步骤。
